Code Structure

What is Code Structure?

    Javascript can be written in 3 code structures:

  • Inline Javascript

  • Internal Javascript

  • External Javascript

Code structure is mainly to say how the Javascript file must be looks like or it will show wheather a external file can be included with ".js" extension or not.

Inline javascript

  • Inline javascript is a techinique in which the attributes of script tag present inside the html itself.

  • The inline javascript reduces the time taken by the browser to serve the request by handling it inside the HTML document

Internal javascript

  • Internal javascript is a techinique in which the attributes of script will be present in between the HTML tags.

  • The internal Javascript is more faster than the external javascript because it handles everything with in the same file.

External javascript

  • External javascript is a techinique in which the attributes of script tag present in a separate /javascriptfile with the extensionof ".js" file.

  • The URL tells the web server where to locate and load the javascript file.

Variables

Generally Javascript is totally different that's why here we didn't have any datatypes to describe the variables.Only we have 2 types:

  • var

  • let

Thus here we can use numbers,strings,boolean,null,undefined.

var

  • The simple difference between the let & var is the scope of the variables.It is also used for declaring a variable and it have scope globally.

  • Syntax:var variablename=value

let

  • The use of let is also used to declare a variable.But it has only local scope.

  • Syntax:var variablename=value

Difference between var & let keyword?

  • If we use "var" keyword for a variable if we haven't declare it it will show undefined.

  • If we use "let" keyword for a variable if we haven't declare it it will show "Refernce Error"

Variable Declaration

  • The variables in javascript can be declared by 3 ways:

    1. camelCase
    2. underscore_case
    3. PascalCase

camelCase

  • camelCase declaration is nothing but here if a variable name has two words,first letter of the second word alone must be in capital letter.
  • Ex:let newName="W4Schools";

underscoreCase

  • underscoreCase declaration is nothing but here if a variable name has two words,underscore will present in between the words.
  • Ex:let new_name="W4Schools";

PascalCase

  • PascalCase declaration is nothing but here if a variable name has two words,both first letter & second word must be in capital letter.
  • Ex:let NewName="W4Schools";
