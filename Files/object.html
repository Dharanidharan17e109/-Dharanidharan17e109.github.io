JavaScriptâ€™s fundamental datatype is the object. An object is a composite value: it aggregates multiple values (primitive values or other objects) and allows you to store and retrieve those values by name. An object is an unordered collection of properties, each of which has a name and a value. Property names are strings, so we can say that objects map strings to values.

Creating Objects

  • SYNTAX: variabletype variablename={property:value};

  • The easiest way to create an object is to include an object literal in your JavaScript code.

  • An object literal is a comma-separated list of colon-separated name:value pairs, enclosed within curly braces.

  • A property name is a JavaScript identifier or a string literal (the empty string is allowed).
