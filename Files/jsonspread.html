JSON Methods

In javascript we have 2 JSON methods:

  • JSON.stringify
  • JSON.parse

JSON.stringify

  • SYNTAX:JSON.stringify(object)

  • JSON.stringify is function which takes an object as an input parameter and convert the into string values in the same type.

  • The resulted object is called as serialized or JSON-decoded Object.

Spread operator

  • SYNTAX: var variableName=[...variableName]

  • The main use of the spread operator is to target all the values in a variable.
