Functions

In javascript we can use functions in 3 ways:

  • Function Declaration
  • Function Expression
  • Arrow Function

Function Declaration

  • Function declartion in javascript means a simple definition of the function like c programming.

Function Expression

  • Here Function Expression is nothing but we will assign a function to a variable and we will access the function using the variable which holds the function definition.

Arrow Function

  • Same as function expression we will declare a function to variable and we will pass the arguments and giving the definition by giving an arrow "=>"
