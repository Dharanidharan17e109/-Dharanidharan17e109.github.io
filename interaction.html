Interactions

In javascript we can interact with the user using 3 functions:

  • prompt
  • alert
  • confirm

prompt

  • prompt function is mainly used to get the input from the user when user is in a webpages.It will provides the prompt window to get an input.


alert

  • alert function is mainly used to alert the user.It will provides the alert window.


confirm

  • confirm function is mainly used to make the user to confirm the task.It will provides the confirm window.
